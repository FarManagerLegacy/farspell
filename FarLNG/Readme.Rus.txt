Библиотека FarLNG 
~~~~~~~~~~~~~~~~~~

версия 1.1 от 11.05.05

1. Описание
~~~~~~~~~~~~

	Библиотека FarLNG предназначена для добавления в плагины/приложения 
возможности читать языковые файлы в формате Far Manager (.lng файлы). 
Библиотека поставляется в виде библиотечного файла (.lib) в формате 
Microsoft Visual Studio .NET с исходными текстами. 

2. Использование библиотеки
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

	Библиотека FarLNG предоставляет следующие функции для загрузки и работы
с .lng файлами:

2.1. LoadLanguageFile
~~~~~~~~~~~~~~~~~~~~~~

bool LoadLanguageFile (
		const char *lpFileName, 
		const char *lpLanguage,
		char **&pStrings, 
		int &nStringsCount
		);


Описание
~~~~~~~~~

	Функция загружает и обрабатывает языковой файл. 

Параметры
~~~~~~~~~~

	lpFileName		- 	полное имя языкового файла.
	
	lpLanguage		-	требуемый язык языкового файла. Этот параметр может быть NULL, 
						если язык языкового файла может быть любым.
	
	pStrings  		-	возвращаемый массив строк из языкового файла.

	nStringsCount	-	количество строк в массиве строк языкового файла. 

Возвращаемое значение
~~~~~~~~~~~~~~~~~~~~~~

	Если функция смогла найти и прочесть языковой файл и его язык соответствует 
языку, переданному в параметре lpLanguage, то функция возвращает true, иначе возвращается
false.

2.2. SearchAndLoadLanguageFile
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

bool SearchAndLoadLanguageFile (
		const char *lpPath,
		const char *lpLanguage,
		char **&pStrings, 
		int &nStringsCount
		);


Описание
~~~~~~~~~

	Функция ищет в заданной папке и загружает языковой файл содержащий строки на требуемом языке.

Параметры
~~~~~~~~~~

	lpPath			-	папка, в которой будет осуществляться поиск языковых файлов.

	lpLanguage		-	требуемый язык языкового файла. Этот параметр может быть NULL, 
						если язык языкового файла может быть любым, в этом случае будет
						загружен первый найденный языковой файл.

	pStrings  		-	возвращаемый массив строк из языкового файла.

	nStringsCount	-	количество строк в массиве строк языкового файла. 

Возвращаемое значение
~~~~~~~~~~~~~~~~~~~~~~

	Если функция смогла найти в заданной папке языковой файл требуемого языка и смогла
его прочесть, то функция возвращает true, иначе возвращается false.

2.3. FinalizeLanguageStrings
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

extern void FinalizeLanguageStrings (
		char **pStrings,
		int nStringsCount
		);

Описание
~~~~~~~~~

	Функция освобождает память, выделенную под массив строк языкового файла.
Эту функцию следует вызывать только в случае успешной загрузки языкового файла
функциями LoadLanguageFile или SearchAndLoadLanguageFile.
	
Параметры
~~~~~~~~~~

	pStrings		-	массив строк языкового файла.

	nStringsCount	-	количество строк в массиве строк языкового файла.

Возвращаемое значение
~~~~~~~~~~~~~~~~~~~~~~

Нет.
