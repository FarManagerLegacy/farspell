    на любом компакте с Visual Studio

    http://www.microsoft.com/whdc/devtools/ddk/orderddkcd.mspx - бесплатно по США.
    (доставка в другие страны $25. Можно заказать сразу 5 копий.)
    eDonkey/BitTorrent (???)

Последовательность сборки:
  0. Установить переменные окружения, например:
    Set MSVCDIR=C:\Program Files\Microsoft Visual C++ Toolkit 2003
    Set PATH=%MSVCDIR%\bin;C:\WINDDK\2600\bin\x86;%PATH%
    Set INCLUDE=%MSVCDIR%\include;%MSVCDIR%\PlatformSDK\include;%INCLUDE%
    Set LIB=%MSVCDIR%\lib;%MSVCDIR%\PlatformSDK\Lib;C:\WINDDK\2600\lib\wxp\i386;%LIB%
  1. Зайти в папку FarSpell
  2a. Запустить "nmake debug"
  [-> В подпапке farspell будет рабочая версия]
  2b. Запустить "nmake release"
  [-> В подпапке farspell будет оптимизированная версия]
  2c. Запустить "nmake full":
  [-> Получится файл farspell-YYYYMMDD.7z - это дистрибутив плагина.]
  [-> Получится файл farspell-YYYYMMDD-debug.7z - это отладочный дистр.]
